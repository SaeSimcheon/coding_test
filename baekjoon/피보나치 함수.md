- 피보나치 수를 구하는 것인데 0 개수를 파악하는 방법에 대해서 생각이 바로나지 않았었음.
- 바로 전 항이 0의 개수.

```python
import sys

sys.stdin = open('input.txt','r')


T=int(sys.stdin.readline())
DP = [float('inf')]*41
DP[0]=0
DP[1]=1
def fibonacci(n) :
        
    if DP[n]!=float('inf') :
        return DP[n]
    else:
        DP[n] = fibonacci(n-1) + fibonacci(n-2) 
        return DP[n]
for _ in range(T):
    given = int(sys.stdin.readline())
    if given == 0 :
        print(1,0)
        continue
    fibonacci(given)
    print(DP[given-1],DP[given])
```
